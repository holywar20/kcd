add-todo-never — Habit

When

Any point a session would otherwise fire add-todo.

Action

Do nothing. Write no entry to any todo list, regardless of the trigger.

Why: some lenses either track deferred work through a different mechanism (a dedicated backlog artifact, an external tracker) or don't want open items tracked at all. This habit exists so that intent can be declared explicitly rather than by every deferred item quietly going untracked because no one wired the habit in the first place.
