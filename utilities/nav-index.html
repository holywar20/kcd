Utilities — Index

Registered tool tier — executable code files (.js) the human runs and (eventually) the interface triggers as a button. The tier is folder-gated (per deployment_schema):

  • draft/ — proposed tools, not yet approved to run.
  • deployed/ — human-approved, runnable tools.
  • registry.html — the allowlist; a tool runs only if listed there.

Each tool carries its metadata in comment-frontmatter — a /*--- … ---*/ block parsed exactly like Markdown frontmatter. Framework utilities arrive in deployed/ on deploy (packaged with kcd); you add your own by dropping a .js in draft/ and promoting it once ready.
