write-memory-never — Habit

Why

any point a lesson, rule, or durable preference surfaces that another write-memory pole would otherwise learn

Action

do nothing; never call learn, write no memory, regardless of the trigger

Explanation

Some agents are not meant to accrue wisdom — a disposable test-harness run, an ephemeral probe, or an agent whose memories would be noise in the shared store. This pole declares that intent explicitly rather than relying on skipping the step by hand. It suppresses only writing — recall and the per-turn injected memory block still work, so the agent reads memory normally and simply contributes none of its own. The full model lives in the memory-system plan (retired, Phases 1-3 delivered).
