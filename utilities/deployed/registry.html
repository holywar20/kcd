Utilities — Registry (canonical)

The allowlist of framework-shipped, approved utilities — packaged with the framework so every project that deploys kcd inherits them. A tool runs only if it is listed here and lives in deployed/. Utilities still in development wait in draft/.

On deploy this registry is copied to the instance (_Claude/utilities/deployed/registry.md), where a project adds its own utilities. Content-addressing (hash + invalidate-on-change + re-approval) is deferred — see Still evolving → Utility tier in kcd_framework.
