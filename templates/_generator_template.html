Generator (template)

Scaffold note

Scaffold for authoring a canonical generator. Copy to _Claude/kcd/generators/{name}/{name}.html (a generator is a folder: the .html trunk plus a context/ subfolder). Holds all generic logic; declares — never solves — its requirements, and names no project-specific path or value. Fill every {placeholder} and delete these scaffold notes.

The authored generator's frontmatter replaces this template's identity fields:

  • type → generator
  • status → active or disabled
  • base → _generator-base

(model is no longer frontmatter — it is the model parameter below.)

Base rules: _generator_base — composition model, requirement resolution, fail behavior, output conventions.
