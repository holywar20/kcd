add-todo-never — Habit

When

any point a session would otherwise fire add-todo

Action

do nothing; write no entry to any todo list, regardless of the trigger

Explanation

Some lenses track deferred work through a different mechanism (a dedicated backlog artifact, an external tracker) or don't want open items tracked at all. This habit exists so intent can be declared explicitly rather than every deferred item quietly going untracked because no one wired the habit in.
