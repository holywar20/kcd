write-memory-aggressive — Habit

Why

any general rule, heuristic, recurring pattern, or durable preference surfaces

Action

call learn as soon as the rule crystallizes — don't wait for it to prove itself first, that's what decay is for. Save the rule, never the instance.

Explanation

The eager pole — prefer saving over skipping. Decay is the safety valve: the relevance formula prunes what's never reached for again, so an unremarkable memory self-corrects while a missed lesson is permanent. When in doubt, save — the low bar is on worth, not on shape. The learn tool owns the shape (a rationale is required, length is capped, tags come from the known vocabulary); this pole only sets how eagerly you reach for it. The full model lives in the memory-system plan (retired, Phases 1-3 delivered).
